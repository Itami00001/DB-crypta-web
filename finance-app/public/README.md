# Finance App Frontend

## Обзор

Frontend для финансового приложения с поддержкой криптовалют, построенный на Bootstrap 5 и Vanilla JavaScript.

## Функциональность

### 🏠 Дашборд
- **Статистические карточки**: отображение количества пользователей, криптовалют, кошельков и транзакций
- **Графики**: рыночная капитализация и распределение транзакций
- **Последние транзакции**: список 5 последних транзакций
- **Популярные криптовалюты**: топ 5 активных криптовалют

### 💱 Криптовалюты
- **Список всех криптовалют**: таблица с детальной информацией
- **Добавление новых криптовалют**: модальное окно с формой
- **Фильтрация**: отображение активных/неактивных монет
- **Статусы**: визуальные индикаторы состояния

### 👛 Кошельки
- **Карточки кошельков**: визуальное представление с балансами
- **Типы кошельков**: горячие, холодные, биржевые, аппаратные
- **Адреса**: отображение полных адресов кошельков
- **Балансы**: точное отображение балансов с 8 знаками после запятой

### 💸 Транзакции
- **Таблица транзакций**: полный список всех транзакций
- **Типы транзакций**: покупка, продажа, перевод, пополнение, вывод
- **Статусы**: в ожидании, завершена, не удалась, отменена
- **Фильтры**: по типу и статусу

### 📰 Новости
- **Карточки новостей**: современный дизайн с превью
- **Типы публикаций**: новости, анализ, обучение, объявления
- **Статусы**: опубликовано/черновик
- **Метрики**: количество просмотров и дата публикации

### 📊 Прогнозы
- **Карточки прогнозов**: цветовая кодировка по типу прогноза
- **Типы прогнозов**: бычий (рост), медвежий (падение), нейтральный
- **Целевые цены**: прогнозируемая и целевая цена
- **Даты**: дата цели и статус активности

## API Интеграция

### Основные эндпоинты
- `GET /api/users` - Получение пользователей
- `GET /api/crypto-currencies` - Получение криптовалют
- `GET /api/crypto-wallets` - Получение кошельков
- `GET /api/transactions` - Получение транзакций
- `GET /api/news-posts` - Получение новостей
- `GET /api/user-predictions` - Получение прогнозов

### CRUD операции
- `POST /api/crypto-currencies` - Добавление криптовалюты
- `POST /api/crypto-wallets` - Добавление кошелька
- `POST /api/transactions` - Добавление транзакции
- `POST /api/news-posts` - Добавление новости
- `POST /api/user-predictions` - Добавление прогноза

## Использование

### Запуск приложения
1. Убедитесь, что backend сервер запущен на порту 6868
2. Откройте в браузере: `http://localhost:6868`
3. Приложение автоматически загрузит данные с API

### Навигация
- **Дашборд**: главная страница со статистикой
- **Криптовалюты**: управление криптовалютами
- **Кошельки**: управление кошельками
- **Транзакции**: история транзакций
- **Новости**: управление новостями
- **Прогнозы**: управление прогнозами

### Добавление данных
1. Перейдите в нужный раздел
2. Нажмите кнопку "Добавить"
3. Заполните форму в модальном окне
4. Нажмите "Добавить" для сохранения
